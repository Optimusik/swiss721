
### 1. Install Dependency

```bash
npm install
```

### 2. Set .env File

create .env file in root project

```bash
PRIVATE_KEY="your private key"
```


### 3. Compile Smart Contract

```bash
npm run compile
```

### 4. Deploy Smart Contract

```bash
npm run deploy
```

### 5. Mint NFT

```bash
npm run mint
```
